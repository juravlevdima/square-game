# Square Game

The project is hosted on Heroku: http://whack-a-square.herokuapp.com

## Backend:
* Node.js
* Express.js
* MongoDB/Mongoose

## Frontend:
* React
* Redux/ReduxThunk
* TailwindCSS
* Axios

## In the project directory, you can run:

### Installation

```Shell
yarn install && yarn run client:install
```
Installs the dependencies.

### Development

```Shell
yarn run dev
```

### Start

```Shell
yarn run client:build && yarn run start
```
